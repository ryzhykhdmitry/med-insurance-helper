# Project Constitution — .NET RAG Chatbot (Study Project)

Purpose
- Minimal study project demonstrating a Retrieval-Augmented Generation (RAG) chatbot implemented in .NET.

Scope
- Learning and local development only. No production deployment or SLA.
- Use free services and local emulators whenever possible.

Primary decisions
- LLM: Foundry (single provider for this project).
- Document storage: Blob storage (prefer free cloud tier).

Testing & QA
- No automated tests required for this study project.

Environment & Secrets
- Prefer local emulators and free tiers. Keep credentials out of source control; use environment variables or local secret stores.

Security & Privacy
- Avoid ingesting sensitive personal data. Redact or exclude sensitive information before ingestion.

Contributors
- Small team or single developer study project.
